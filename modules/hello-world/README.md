# hello-world

This module creates a lambda that prints a nice message to the world.

## Usage

To use this module use the example in the [main.tf](main.tf)

```terraform
module "hello-world" {
  source = "github.com/danto7/sda_coding_challenge?ref=v0.0.0"

  cron_expression = "* * ? * * *"
}
```
